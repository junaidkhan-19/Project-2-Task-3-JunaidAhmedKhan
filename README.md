# ⚡ Energy Consumption Time Series Forecasting

## 📌 Project Overview

This project focuses on forecasting short-term household energy consumption using historical power usage data. The workflow includes data preprocessing, time series resampling, feature engineering, and the implementation of multiple forecasting models. The models are evaluated using standard regression metrics to identify the best-performing approach.

## 🎯 Objectives

- Parse and preprocess household energy consumption data.
- Resample the time series for forecasting.
- Engineer time-based features (Hour, Day, Month, Weekday/Weekend, etc.).
- Build and compare ARIMA, Prophet, and XGBoost forecasting models.
- Evaluate model performance using MAE and RMSE.
- Visualize actual vs. forecasted energy consumption.

## 📈 Evaluation Metrics

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

## 📂 Dataset

## 🚀 Outcome

The project compares three forecasting techniques and identifies the model with the lowest prediction error for short-term household energy consumption forecasting.
